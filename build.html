Setup your Pi

I am going to assume that you have the Raspberry Pi configured and working, however here are the other assumptions:

  1. Raspberry Pi setup and up to date
  2. Configured for headless access (this will be easier to manage than eneding to connect a screen and keyboard etc
  3. External USB storage is mounted to save files to. (This means we can save the SD card from getting full of images)

Webcam

I based most of the webcam setup on the guide here, but again customised some bits

Install the basic python code form this site, then we can set about editing it:

sudo apt-get install python-imaging-tk
wget -c http://pastebin.com/raw.php?i=yH7JHz9w -O picam.py
chmod +x picam.py

I decided not to show a large mount of images on the system, so instead am saving a latest image down and displaying that.
